What Parts Are In A Computer?

Overall List

  • CPU
  • Motherboard
  • RAM
  • Powersupply
  • Hard Drive
  • GPU
  • Fans
  • Case
  • Operating System

CPU

The CPU, or Central Processing Unit, is the heart of a computer. This is where all of the bits of information get manipulated. Two major companies who manufature CPUs are Intel and AMD.

Motherboard

The motherboard is where every component of the computer comes together. Each piece connects to the motherboard so that they can interact with each other.

RAM

The RAM is the short-term memory of the computer. Data is temporarily stored there so that it can be quickly accessed. But once the computer is turned off, the memory on the Ram is wiped

Powersupply

The Powersupply is very self explanatory. This unit sends power to every piece in the computer.

Hard Drive

The Hard Drive is the long-term memory of the computer. Data is permenately stored here, but accessing it is much slower than accessing data from the RAM

GPU

The GPU, or Graphical Processing Unit, is an extra piece to help the CPU. This part is dedicated to help process the graphical images seen on a monitor.

Fans

Fans help to keep the parts of the computer cool. There is always a dedicated fan for the CPU and the GPU that is attached with a heat spreader.

Case

The case is the unit you store all of the parts in.

Operating System

The operating system is the program you use to run all of the parts together. The two main operating systems are Windows and OSX
